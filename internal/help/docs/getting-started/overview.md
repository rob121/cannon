# Overview

Welcome to **Cannon** admin help.

Cannon is a multi-site CMS that loads extensions as separate processes and routes requests through a configurable middleware chain.

## Key areas

- **Content** — routes and menus for the public site
- **Extensions** — optional processes that add pages, blocks, admin UIs, and help docs
- **System** — users, sites, languages, and extension process registry

Use the sidebar to browse topics by folder. Extension-specific documentation appears under **Extensions** when an installed extension publishes help articles.
